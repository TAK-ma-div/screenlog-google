# 行動規範 (Code of Conduct)

本プロジェクトは [Contributor Covenant](https://www.contributor-covenant.org) v2.1 を採用します。

## 私たちの誓い

私たちメンバー、コントリビューター、リーダーは、年齢・体格・心身の障害・民族・
性自認や性表現・経験の有無・国籍・容姿・人種・宗教・性的指向などにかかわらず、
誰もがハラスメントのないコミュニティ体験を得られるよう努めます。

## 行動の基準

コミュニティにとって望ましい行動の例:

- 他者への共感と思いやりを示す
- 異なる意見・視点・経験を尊重する
- 建設的なフィードバックを与え、また快く受け入れる
- 自らの過ちに責任を持ち、影響を受けた人々に謝罪し、その経験から学ぶ
- 個人ではなくコミュニティ全体にとって最善であることに目を向ける

容認されない行動の例:

- 性的な言動や、あらゆる種類の性的な注目・誘いかけ
- 侮辱・中傷的なコメント、個人的・政治的な攻撃（トローリング）
- 公的・私的を問わないハラスメント
- 明示的な許可なく他者の個人情報（住所・メールアドレス等）を公開すること
- 職業上不適切と合理的に判断されるその他の行為

## 執行の責任

プロジェクトのメンテナは、行動の基準を明確化・執行する責任を負い、不適切・
脅迫的・攻撃的・有害とみなした行為に対して、公正かつ適切な是正措置を取ります。

## 適用範囲

本行動規範は、プロジェクトのすべての空間に適用され、また個人がコミュニティを
公的に代表する場面（公式メールアドレスの使用、公式SNSアカウントでの投稿、
オンライン/オフラインのイベントでの代表など）にも適用されます。

## 執行

虐待・ハラスメント・その他の容認されない行動は、プロジェクトのメンテナ
（GitHub: [@TAK-ma-div](https://github.com/TAK-ma-div)）まで、可能な限り
非公開の手段（GitHub の Security Advisories など）でご報告ください。
すべての申し立ては速やかかつ公正に検討・調査されます。
メンテナは、報告者のプライバシーと安全を尊重します。

## 帰属

本行動規範は [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
バージョン2.1から引用しています。
